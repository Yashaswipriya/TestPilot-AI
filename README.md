# TestPilot AI

AI-powered test generation for GitHub repositories.

## Stack
- **Frontend**: Next.js (App Router) + TypeScript + Tailwind CSS + shadcn/ui
- **Backend**: Express.js + TypeScript + MongoDB

## Structure
- `client/` — Next.js frontend
- `server/` — Express.js backend

## Getting Started

### Client
```bash
cd client
npm install
npm run dev
```

### Server
```bash
cd server
npm install
npm run dev
```
